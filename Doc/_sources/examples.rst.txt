Examples
#############

The directory ``examples/`` in TBSBIEM Documentation' root repository contains example scripts
dedicated to various aspects of TBSBIEM:

USGC/SCEC Benchmark Problems
*********************************

We have solved benchmark problems TPV3, TPV5, TPV6 and TPV7 using the TBSBIEM code. 

Crack Front Wave Analysis
***********************************
We have used the base code and made a compatible code to understand the behavior of crack front waves under mode-II and Mixed mode with mode-II and III.
